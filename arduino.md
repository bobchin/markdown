** メモ

- Arduino Uno : Arduino Duemilanove の後継

```
最大定格電流：LEDに最大定格以上の電流を流し続けるとLEDは壊れる
電気的特性：LEDに電気的特性の順電流を流したら、LEDの電圧が電気的特性の順電圧になる
順方向電流：IF
順方向電圧：VF
```
