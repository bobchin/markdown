## 使い方

- ショートカットキー

| キー      | 内容                   |
| :-------- | :--------------------- |
| ⌘ + D     | ペインを左右に分ける   |
| ⇧ + ⌘ + D | ペインを上下に分ける   |
| ⌘ + W     | ペインを閉じる         |
| ⌘ + P     | コマンドパレットを開く |
| ⇧ + ⌘ + S | パーマリンクを作成 |
| ^ + ⌘ + R | worlflowを開く |

- Blocks
  - コマンド入力と出力を１つのBlockという単位とする。

- worlflow
  - Warp 内でコマンドを実行して共有する機能
  - ユーザスニペットみたいな感じ？
