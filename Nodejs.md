# Node.js

## nvm

インストール

```
cd 
git clone git://github.com/creationix/nvm.git
```

> ~/.zshrc.local

以下を追記
```
export NODE_PATH=${NVM_PATH}_modules
. ~/nvm/nvm.sh
```

```
nvm ls-remote
nvm install v0.9.3
nvm alias default v0.9.3
```
